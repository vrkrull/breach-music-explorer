# breach-music-explorer
 Breach music explorer is a music explorer GUI that searches up music from various databases.
 When you click a track, it plays a MUSIC PREVIEW, NOT THE FULL TRACK - THIS IS TOTALLY LEGAL.

## How to get this app started

THIS APP IS CROSS-PLATFORM IT WILL RUN WITH ANYTHING WITH PYTHON AND PYTHON LIBRARIES/pip ON IT
How to install python: https://realpython.com/installing-python/
How to install pip: https://pip.pypa.io/en/stable/installation/
Once you've installed python and pip, download the code and run "python main.py" in the terminal.
If any errors pop up about a moudle not being installed, run "pip install <moudle>"
